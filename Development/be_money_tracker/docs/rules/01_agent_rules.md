# AI AGENT RULES

## MUST
- Đọc toàn bộ docs trước khi code
- Tuân thủ architecture + database + API đã định nghĩa

## MUST NOT
- Không tự ý code khi chưa có doc
- Không tự ý sửa DB structure
- Không tự ý đổi API

## WHEN DOING TASK
- Nếu thiếu thông tin → yêu cầu user update docs
- Nếu thay đổi logic → ghi vào decision-log.md

## OUTPUT
- Code phải match với docs 100%
